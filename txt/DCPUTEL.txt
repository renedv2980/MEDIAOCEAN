         TITLE 'PUTEL'                                                          
Inserts an element into a record in sequence with the element dictating its own 
sequence.                                                                       
                                                                                
Parameter 1                  A(record)                                          
                                                                                
Parameter 2                  A(element)                                         
                                                                                
After PUTEL, Parameter 1 will contain the new length of the record.             
                                                                                
PUTEL will blow up if if finds zero length elements.                            
                                                                                
Note - For PUTEL to account file, include ACPUTEL.
