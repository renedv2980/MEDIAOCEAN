         TITLE 'EXMVC and EXCLC'                                                
Macro allows user to write a variable lenth move or compare instruction in one  
statement.                                                                      
                                                                                
     EXMVC RN,Afield,Bfield                                                     
     EXCLC RN,Afield,Bfield                                                     
                                                                                
RN - contains length code (L'Afield-1).
