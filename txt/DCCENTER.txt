         TITLE 'CENTER'                                                         
Module will centralize a string of data with trailing blanks, passing back      
a string with both leading and trailing blanks.                                 
                                                                                
Parameter 1                       A(input)                                      
                                                                                
Parameter 2                       L'input                                       
                                                                                
 Example                                                                        
 -------                                                                        
                                                                                
Passed   - C'CENTRALIZE THIS         ' - length=25                              
                                                                                
Returned - C'     CENTRALIZE THIS     '                                         
                                                                                
The on-line, core-resident version is T00A01.
